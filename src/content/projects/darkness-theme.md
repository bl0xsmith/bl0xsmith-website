The theme you're currently viewing. Darkness is an experimental Astro theme built around the idea that a portfolio should feel like an experience — not just a page.

## Features

- 5000 Three.js particles rendered in 3D space
- Custom cursor with interactive hover effects
- Glitch text animations using CSS custom properties
- Smooth page transitions via the View Transitions API
- Full TypeScript support throughout
- Blog and projects collection via Astro Content Collections
- Zero-JS fallback for core content

## Design Philosophy

The aesthetic is intentional — deep blacks, neon accents, and motion that responds to the user. Every animation is purposeful: particles drift, text glitches on entry, and transitions carry context between pages.

## Tech Stack

Built with Astro for near-zero JS overhead at the framework level, Three.js for the 3D particle canvas, and vanilla TypeScript for all interaction logic. No CSS framework — every style is hand-written.
