Void is a VS Code color theme that leans into true black backgrounds, muted syntax colors, and neon accents only where they matter. No gradients, no noise — just the code.

## Design Goals

Most dark themes are dark grey, not dark. Void uses `#000000` as the base and builds up from there with carefully chosen token colors that reduce eye strain during long sessions without sacrificing readability.

## Color Palette

- Background: `#000000`
- Surface: `#0a0a0a`
- Comments: desaturated purple-grey
- Strings: soft green
- Keywords: neon violet
- Functions: muted cyan
- Errors: deep red — visible but not aggressive

## Installation

Search **"Void"** by bl0xsmith in the VS Code Extensions marketplace, or install via the CLI:

```bash
code --install-extension bl0xsmith.void-theme
```
