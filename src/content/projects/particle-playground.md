A browser-based sandbox for building and experimenting with Three.js particle systems — no setup, no build step, just open and tweak.

## What You Can Do

Dial in particle count (up to 50,000), color gradients, spawn shapes, velocity curves, and gravity with live sliders. Every change reflects instantly on the canvas. When you land on a configuration you like, export it as a JSON preset or grab a screenshot of the current frame.

## Highlights

- Real-time parameter adjustment with zero lag
- Multiple emitter shapes: sphere, box, cone, disc
- Physics options: gravity, turbulence, drag
- Color palette picker with gradient support
- Preset export / import as JSON
- High-res screenshot capture
- Works entirely in the browser — no server required

## Why I Built It

I kept copy-pasting particle config code between projects. This turns that iteration loop into a visual tool. Design the system here, export the config, drop it into any Three.js project.
